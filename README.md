Hybrid Reinforcement Learning with an Avellaneda–Stoikov Backbone

A structured reinforcement learning framework for high-frequency market making in limit order books, combining the analytical Avellaneda–Stoikov (AS) model with a data-driven residual reinforcement learning controller.

This repository implements the methodology presented in the paper:

Hybrid Reinforcement Learning with an Avellaneda–Stoikov Backbone for Order-Book-Aware High-Frequency Market Making under Non-Stationary Microstructure
